## Выпускной проект №2 - "Напишите свою реализацию MVC"

##### For run on Windows:
1. OpenServer -> Settings -> Domains -> add {"Domain name": "mvc", "Domain folder": "\phpls-022020-final-work-2\public"}
2. Composer update
3. Create DB & DB_User (/database/dump.sql)
3. Go to http://mvc/
